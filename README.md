###  <a id="index">简介</a>
SKCMS是由<a href="https://github.com/FisH-2022/">fish</a>开发的内容管理系统，可用于<code>搭建博客、官网等</code>。SKCMS
可以安装任意适配SKCMS的主题，也欢迎各位开发者为SKCMS开发主题。本主题采用MIT开源协议，欢迎二次开发，只需<code>在明显处留下原作者及此仓库链接</code>即可~<br>

### 📌<a>功能优势</a>
* [x] 所有功能免费
* [x] 主题
  * [x] 自带默认主题
  * [x] 主题适配简单
  * [x] 可安装自定义主题
  * [x] 主题商店超多主题
  * [x] 主题开发支持
* [x] 全站用户管理
* [x] 站点信息统计
* [x] 富文本编辑器



### 🛠 <a id="install">安装要求</a>
- Nginx / Apache / IIS
- PHP 7.0 / 7.0+
- Mysql 选择任意稳定版本即可
